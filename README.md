# Pygame Projects

A collection of small games made with **Python** and **Pygame**.

## Games

### Flappy Bird

A simple clone of the Flappy Bird game.

Run the game:

```
cd "Flappy bird"
python game1.py
```

More games will be added in the future.

## Requirements

Install pygame:

```
pip install pygame
```
